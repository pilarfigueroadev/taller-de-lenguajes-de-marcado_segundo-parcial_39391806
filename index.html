Reglamento

Código de Conducta:

Se espera que todos los participantes mantengan una actitud de respeto y deportividad. No se tolerará ningún tipo de acoso, discriminación o toxicidad.

Fair Play:

Está estrictamente prohibido el uso de trampas, hacks o cualquier software de terceros que otorgue una ventaja injusta. Las cuentas smurf no están permitidas.

Inscripción:

Para participar, completa el formulario de inscripción y realiza la donación mínima sugerida a la causa del torneo. El 100% de lo recaudado se destina a la ONG beneficiaria.
